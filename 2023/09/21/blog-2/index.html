    next 主题-标签
    Hexo 标签

    title: Tag Plugins
    description: NexT User Docs – NexT Supported Tags

    标签插件是一种制作 Hexo 支持的特殊样式内容的方法。例如，我们无法在标准 Markdown 中显示具有自定义大小的图像。然后我们可以使用标签插件来解决它。Hexo有很多标签Hexo has a lot of tags这可以帮助用户。Hexo还具有主题接口，使主题能够创建自己的标签。NexT提供以下标签：

    • Button
    • Caniuse
    • Centered Quote
    • Group Pictures
    • Label
    • Link Grid
    • Mermaid
    • WaveDrom
    • Note
    • PDF
    • Tabs
    • Video

    These tag plugins are only available in the theme NexT. If you switch to other Hexo themes, they may cause rendering errors.

    Centered Quote

    此标签将在其前后各有两行引用，引用的文本将居中。使用居中引号时，如果我们有多行文本，并且每行都有不同的长度，则引用不会对称，因此建议在只有单行文本时使用。例如在文章之前所有文章之后做一个总结。
